Tensile Test Analyzer
---------------------

How to Run:
1. Ensure Python 3.8+ is installed.
2. Double-click "run_app.bat".
   This will install required packages (First time only) and launch the server.
3. Your browser should open automatically at http://127.0.0.1:5000.

Features:
- Upload CSV files containing "Displacement" and "Load (N)".
- Automatic Stress-Strain Curve generation.
- Calculation of Max Stress, Max Strain, Max Load.
- Visualization of Elastic Region (Blue area on Displacement-Load chart).
- Fracture Point Detection (Red Cross).
- Tooltip displays Instantaneous Slope Angle on hover.
- Retro-themed UI (Pink/Dark Mode).

Troubleshooting:
- If the browser doesn't open, manually go to http://127.0.0.1:5000 in Chrome/Edge.
- If errors occur, ensure Python is in your system PATH.
